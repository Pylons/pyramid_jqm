About pyramid_jqm

Home

Pyramid is a small, fast, down-to-earth web framework written in Python. pyramid_jqm is a scaffolding package which gets you started on a simple jQuery Mobile application.

This application is using pyramid_jqm version and Pyramid version .

The version information was retrieved via an XHR request to the Pyramid application, which returns JSON.
