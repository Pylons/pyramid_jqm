pyramid_jqm
===========

Overview
--------

``pyramid_jqm`` is a package which provides "starter" scaffolding for
creating a :term:`jQuery Mobile` Pyramid application.

Installation
------------

Install using setuptools, e.g. (within a virtualenv)::

  $ easy_install pyramid_jqm
